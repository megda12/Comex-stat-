## OBJETIVO DO PROJETO
O objetivo é identificar os principais recursos, processos e estruturas do sistema produtivo. A proposta busca avaliar a eficiência operacional
com base em indicadores como capacidade instalada, tempos de ciclo, produtividade, níveis de conformidade e utilização dos recursos, considerando seu
papel estratégico na competitividade e melhoria contínua.

Para a consecução deste objetivo foram estabelecidos os objetivos específicos:
•	Dashboard interativo no Power BI
•	Decodificar os dados no Phyton
•	Ter informações referente a 3 Municípios no mínimo
•	Ter indicadores e gráficos que sejam interativos e claro
•	Usar dados abertos do Comex Stat
